INTRODUCTION
------------

Axelerant module divided into two phase.
1. Alter the site information form.
2. Provides a URL that responds with a JSON representation 
 of a given node with the content type "page".


INSTALLATION
------------

The axelerant module is very similar to other Drupal modules.
For installation of the axelerant module please follow the 
 below mentioned steps:

• Install as usual
• Enable the axelerant module.


DEPENDENCY
----------

• There is no dependency.


CONFIGURATION
-------------

• There is no configuration for this module.
